Vue踩坑日记

基本指令：

v-model： 指令用来在 input、select、text、checkbox、radio 等表单控件元素上创建双向数据绑定，根据表单上的值，自动更新绑定的元素的值。按钮的事件我们可以使用 v-on 监听事件，并对用户的输入进行响应。

v-html： 指令用于输出 html 代码格式

v-bind : 指令被用来响应地更新 HTML 属性

v-if:条件判断使用 v-if 指令

生成一个随机数 判断大于0.5

v-fo：循环使用 v-for 指令。http://www.runoob.com/vue2/vue-loop.html

v-for 指令需要以 site in sites 形式的特殊语法， sites 是源数据数组并且 site 是数组元素迭代的别名。
